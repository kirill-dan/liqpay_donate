Данный модуль поможет организовать следующий функционал:

Установка в любом месте сайта блока, в котором можно совершить пожертвования. А именно, пользователь указывает свой имя, свой номер телефона, e-mail адрес, сумму пожертвования, которое он хочет совершить для данного сайта, и выбрать платежную систему, через которую он сможет осуществить пожертвование. 
После нажатия на кнопку «Пожертвовать», пользователь будет переправлен либо на страницу благодарности (если платеж осуществляется в автоматическом режиме), либо вначале на сайт платежной системы для совершения платежа, а уже после этого на страницу благодарности.

В админ меню, в разделе «Configuration” будет пункт «Donate settings” при клике на который будет выведена форма настроек для управления платежами и платежными системами.

В левой колонке формы будут выводится вкладки, каждая из которых соответствует своей платежной системе.

В первой версии модуля будет доступна пока только одна платежная система — LiqPay.

При выборе платежной системы будут доступны поля для редактирования настроек платежной системы:

LiqPay:

Title – текст с кратким описанием, что это за платежная система
Signature — сигнатура платежной системы LiqPay (выдается платежной системой)
Merchant_id — идентификационный номер в платежной системе (выдается платежной системой)
Site URL — ссылка на которую осуществится перенаправление пользователя при клике на кнопку «Donate»
Active — если отмечен этот чекбокс, то система является активной и будет доступна при выборе в блоке во время совершения пожертвования.

Нижняя часть формы настроек содержит закладку «Thanks text». Здесь в текстовое поле можно поместить текст благодарности, который будут выводиться пользователю после выполненного им пожертвования.

В последующих версиях модуля функционал и настройки будут расширяться. 

Для анализа совершенных пожертвований должна существовать статистика. Статистика доступна в админ меню в разделе «Reports» в пункте «Donate report».
Данная статистика по сути является таблицой (List of those who made a donation). Данная таблица имеет следующие поля:
Name – имя человека, которое он ввел при совершении пожертвования
Amount UAH – сумма совершенного пожертвования в украинских гривнах
Status – в каком статусе сейчас находится платеж. Not paid – не оплачен, Paid – оплачен
Payment system – платежная система, через которую совершался платеж
E-mail – e-mail адрес пользователя, который он ввел при совершении пожертвования
Phone – номер телефона пользователя, который он ввел при совершении пожертвования
Date – дата совершения пожертвования

В таблице можно делать сортировку по любым полям. Для этого можно кликнуть на заголовок поля и сортировка будет осуществляться по этому полю. При этом можно сортировать как по возрастанию, так и по убыванию.

В последующей версии модуля планируется добавить еще платежных систем и сделать адресную книгу. Адресная книга может содержать людей или организации, для которых можно настраивать отдельно процесс пожертвований.
